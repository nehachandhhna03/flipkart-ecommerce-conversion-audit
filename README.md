# 🛒 Flipkart E-Commerce Conversion Audit

## 📌 Project Overview
This project is a simple Python GUI application developed using the Tkinter library. It performs a basic UX and conversion audit of the Flipkart e-commerce website by highlighting strengths, weaknesses, and recommendations for improving user experience and conversion rates.

## 🎯 Objectives
- Analyze the homepage user experience.
- Review the product page design.
- Evaluate the checkout process.
- Provide recommendations to improve conversions.

## ✨ Features
- Homepage Audit
- Product Page Audit
- Checkout Audit
- UX Recommendations
- Simple GUI built using Python Tkinter

## 🛠️ Technologies Used
- Python 3
- Tkinter

## ▶️ How to Run
1. Install Python 3.
2. Download the project files.
3. Open the project folder.
4. Run the following command:

```bash
python "flipkart audit.py"
```

## 📷 Output
The application opens a graphical interface where users can click different buttons to view the Flipkart UX audit and recommendations.
